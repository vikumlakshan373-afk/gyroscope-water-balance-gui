# gyroscope-water-balance-gui
A Python-based desktop GUI application for monitoring and controlling a gyroscope water balancing system. Built with Tkinter and ttkbootstrap, it features RPM and angle meters, water pump control, countdown timer, tube status monitoring, input validation, and JSON-based configuration storage.
